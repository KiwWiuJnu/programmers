# 📝 [Programmers / Level 2] 주식가격

> **분류**: 스택/큐  
> **문제 링크**: [주식가격](https://school.programmers.co.kr/learn/courses/30/lessons/42584)

## 1. 문제 파악 및 제한 조건
* **핵심 목표**: 매 초마다 기록된 주식가격 배열이 주어질 때, 각 시점의 가격이 떨어지지 않고 유지된 기간(초)을 계산하는 것입니다.
* **데이터 범위 및 제약 사항**:
  * 주식 가격은 1 이상 10,000 이하 자연수.
  * `prices` 배열의 길이는 2 이상 100,000 이하입니다.
  * $O(N^2)$ 풀이는 최악의 경우 $100,000^2 = 10,000,000,000$ (100억 번 연산)으로 시간 초과 가능성이 있습니다. 효율적인 알고리즘이 필요하며, 목표는 $O(N)$ 시간 복잡도 내에 해결하는 것입니다.

---

## 2. 논리 전개 및 풀이 과정 (Approaches)
### 🚨 1차 시도 (원시적인 brute force 방식)
* **초기 설계**: 
  * 모든 가격에 대해 이중 반복문을 돌며 현재 가격보다 작은 가격이 나올 때까지 탐색합니다.
  * `prices[i]`에 대해 `j = i + 1`부터 돌며 `prices[i] > prices[j]`인 순간까지 카운트를 증가시킵니다.
* **한계점**:
  * 구현은 매우 단순하고 직관적이지만 시간 복잡도가 $O(N^2)$입니다.
  * 평균적으로는 빠를 수 있지만, 최악의 경우(예: 주식이 계속 오르는 경우) 내부 루프가 매번 끝까지 돌아 시간 초과를 받게 됩니다.

### ✅ 2차 시도 (스택을 활용한 최적화 및 최종 통과)
* **개선된 논리**:
  * 스택을 사용하여 아직 가격이 떨어지지 않은 시점(인덱스)을 보관합니다.
  * 매 주식 가격을 순회하며, 스택에 저장된 최상단 값(이전 가격)보다 현재 가격이 낮다면(가격이 하락했다면), 가격이 떨어진 시점입니다.
  * 스택에서 가격이 떨어진 시점들을 꺼내며 유지 기간(`현재 인덱스 - 이전 인덱스`)을 계산해 기록합니다.
  * 이 과정은 스택이 비거나 이전 가격이 현재 가격보다 작거나 같을 때까지 반복합니다.
  * 순회가 끝나고 스택에 여전히 남아있는 인덱스들은 끝까지 가격이 떨어지지 않은 주식들입니다. 이들에 대해 전체 길이에서 해당 인덱스를 뺀 값(`(전체 길이 - 1) - 인덱스`)을 기록합니다.
* **결과**:
  * 모든 원소는 스택에 정확히 1번 들어가고 1번 나옵니다.
  * 시간 복잡도가 $O(N)$으로 획기적으로 단축되어 효율성 테스트까지 모두 통과하였습니다.

---

## ⏱️ 3. 복잡도 분석 및 성능 비교
* **시간 복잡도**: $O(N)$ ➡️ 루프를 한 번 순회하며 각 요소를 스택에 `push`와 `pop`을 최대 1번씩만 수행하므로 전체 연산 횟수는 $N$에 비례합니다.
* **공간 복잡도**: $O(N)$ ➡️ 스택을 위한 추가적인 공간이 최대 $N$개 소요됩니다.

## ⚠️ 4. 엣지 케이스 및 실수 (Mistakes & Edge Cases)
* **스택 언더플로우 (Segmentation Fault)**:
  * 내부 `while` 루프에서 스택의 원소를 꺼내며 이전 값들과 비교할 때, 스택이 비어있는 상태에서 `st.top()`을 참조하는 실수가 있었습니다.
  * `while` 조건에 `!st.empty()`를 선행하여 빈 스택 참조 오류를 방지해야 합니다.
* **최후의 잔여 처리**:
  * 끝까지 가격이 떨어지지 않은 주식들은 루프가 끝난 뒤에도 스택에 남아있습니다. 이들을 올바르게 털어내야 최종 결과가 올바르게 나옵니다.

## 🧠 5. 핵심 디테일 및 딥다이브 (Key Takeaways)
* **단조 스택 (Monotonic Stack)**:
  * 값이 증가하는 성향(혹은 감소하는 성향)을 유지하도록 스택을 제어하는 단조 스택 기법을 사용했습니다.
  * 특정 조건을 만족할 때까지 이전 데이터들을 역추적하며 일괄 처리해야 하는 문제에 스택이 강력한 도구가 됨을 알 수 있습니다.
* **최종 코드 블록**:
```cpp
#include <bits/stdc++.h>

using namespace std;

vector<int> solution(vector<int> prices) {
    vector<int> answer(prices.size());
    stack<pair<int, int>> st;

    for (int i = 0; i < prices.size(); i++) {
        while (!st.empty() && st.top().second > prices[i]) {
            answer[st.top().first] = i - st.top().first;
            st.pop();
        }
        st.push({i, prices[i]});
    }

    while (!st.empty()) {
        answer[st.top().first] = (prices.size() - 1) - st.top().first;
        st.pop();
    }

    return answer;
}
```
